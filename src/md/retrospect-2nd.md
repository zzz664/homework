# 두 번째 회고

- **2주차에 배운 내용들**
  - **tag**  
    `<input>`, `<select>`, `<textarea>`, `<dialog>`, `<details>`, `<summary>`, `<script>`, ...

  - **attribute**  
    `popover`, `popover-target`, `popover-trigger`  
    그리고 여러가지 전역속성(`id`, `class`, `style`, ...)

  - **CSS**
    - **Syntax**  
      selector { property: value; property: value; ... }  
      위와 같은 형식의 문법임
      주석은 `/* 주석내용 */`의 형태로 작성이 가능하다.  
      이외에도 상대단위, 절대단위, 색상값, 스타일시트를 이용하는 방법 3가지 등을 배움

    - **Selector**  
      **전체** 선택자 -> `*`  
      **요소** 선택자 -> `p`, `body` 등 HTML 태그이름을 사용한다.  
      **클래스** 선택자 -> `.ㄴclassName`, `.example` 등
      - 클래스이름 앞에 `.`을 붙여 사용한다.

      **아이디** 선택자 -> `#id`, `#example` 등
      - 아이디이름 앞에 `#`을 붙여 사용한다.

      **속성** 선택자 -> `[title]`, `[example]` 등
      - `[속성이름]`의 형태로 사용한다.

      **자손 결합자** -> 2개 이상의 선택자를 조합하여 가장 마지막 선택자의 조상이 존재하면 해당 선택자에 스타일을 적용한다.
      - `.className p { ... }`의 경우 `p`의 조상 중 `className`클래스를 가진 조상이 있으면 `p`태그에 스타일을 적용한다.

      **자식 결합자** -> 2개 이상의 선택자를 조합하여 가장 마지막 선택자의 부모 요소가 존재할 경우 해당 선택자에 스타일을 적용한다.
      - `.className > p { ... }`의 경우 `p`의 직계 부모 요소가 `className`클래스를 가지고 있을 때 `p`태그에 스타일을 적용한다.

      **선택자 리스트** -> 여러 개의 선택자를 `,(쉼표)`로 구분하여 동일한 스타일을 적용할 수 있음
      - `h1, h2, h3, h4, h5, h6 { ... }`의 경우 `h1~6`태그에 모두 동일한 스타일을 적용한다.

      **가상 요소** -> 선택자에 추가하는 키워드이다.
      - `::before`의 경우 선택자의 앞에 가상의 `span`태그 느낌의 가상 요소를 삽입하여 스타일을 적용할 수 있음
        `::after`의 경우는 선택자의 뒤쪽에 해당한다.  
        이외에도 많은 가상 요소가 있음

      **가상 클래스** -> 선택자에 추가하는 키워드이다. 선택자로 선택한 요소가 특별한 상태일 때라는 조건부 스타일을 적용할 수있다.
      - `:hover`의 경우 마우스가 해당 선택자로 선택된 요소위에 올라간 상태를 뜻한다.
        `p:hover { ... }`라면 `p`태그에 마우스가 올라갔을 때 적용되는 스타일임  
        이외에도 많은 가상 클래스가 있음

    - **Inheritance(상속), Cascade(겹침), Specificity(우선순위)**  
      해당 파트에선 스타일이 어떤 규칙에 따라 적용되는지 살펴보았음

    - **Typography**
      해당 파트는 아주 살짝 진도나감
